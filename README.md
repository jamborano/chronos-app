# ⏱️ Chronos - Pomodoro Timer Premium

Chronos adalah aplikasi timer Pomodoro premium dengan fitur statistik, alarm keren, mode bebas iklan (VIP), dan pembayaran via Midtrans (GoPay/QRIS).

## ✨ Fitur

- 🍅 Timer Pomodoro (25/5/15 menit)
- 📊 Statistik harian & mingguan
- 🔔 Alarm premium
- 🚀 Mode VIP (tanpa iklan, input tugas, simpan data)
- 💳 Pembayaran via Midtrans (GoPay/QRIS)
- 🔐 Login dengan Google (Supabase Auth)
- 🌗 Dark / Light mode
- 🎵 Musik background (YouTube auto-play)

## 🛠️ Teknologi

- **Frontend**: Next.js 15, React, TypeScript, TailwindCSS
- **Backend**: Supabase (Auth + Database)
- **Payment**: Midtrans (GoPay/QRIS)
- **Deployment**: Vercel

## 📦 Instalasi

```bash
# Clone repository
git clone https://github.com/jamborano/chronos-app.git
cd chronos-app

# Install dependencies
npm install

# Setup environment variables (copy .env.local.example to .env.local)
cp .env.local.example .env.local

# Jalankan development server
npm run dev
